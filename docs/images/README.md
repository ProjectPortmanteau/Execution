# Platform Screenshots

This directory contains visual documentation for the OPVS platform.

## Image Files

The following screenshots should be placed in this directory:

1. **boolean_bean_forge.png** - "Boolean / Bean Forge" showing Fluid Mode / Crystallization interface
2. **forge_a_bean.png** - "Forge a Bean" showing the Bean Creation Interface
3. **immune_connectome.png** - "Immune Connectome (BBI)" showing the Force Graph visualization
4. **the_signal.png** - "The Signal" showing the Landing Page

## Usage

These images are referenced in the main README.md in the "Platform Preview" section.

Images should be:
- PNG format (preferred for UI screenshots)
- High resolution (at least 1280px wide for primary screenshots)
- Properly cropped to show relevant UI elements
- Named using snake_case convention as shown above
